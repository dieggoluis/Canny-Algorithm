# Canny edge detector

Implementation of the canny algorithm for detecting a wide range of edges in images.

## Create executable
```sh
$ cmake src/CMakelist.txt
$ make -f <path_to_makefile>
```
## Run Canny edge detection algorithm
```sh
$ ./canny <path_to_image>
```
